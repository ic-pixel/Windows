# Windows
Windows 97 - 11 repository

Windows CMD – Comandos Administrativos (Win 10, Win 11)

Repositório com comandos úteis do **Prompt de Comando (CMD)** executados
com permissões de administrador no Windows.

Este material é focado em **gerenciamento de usuários, permissões e tarefas básicas**
utilizadas em suporte técnico e administração de sistemas.

 Observação:
Todos os comandos devem ser executados com o **CMD aberto como administrador**.

 Conteúdo:
- Gerenciamento de usuários
- Senhas e grupos
- Informações do sistema
- Comandos administrativos úteis

 Objetivo:
 Consulta rápida
